# AGENTS.md

You are a senior full-stack engineer. Build a complete, production-ready MVP without asking for confirmation unless absolutely necessary.

## Project

Build **ActionPilot**, an AI Productivity Companion.

## Tech Stack

* Next.js + TypeScript + Tailwind CSS
* FastAPI (Python)
* Supabase (Database & Auth)

## Requirements

* Responsive landing page and dashboard.
* Users can sign in, create, edit, delete and complete tasks.
* Store all data in Supabase.
* Each task contains title, description, priority, deadline and available hours.
* Implement task prioritization, scheduling, deadline risk and productivity recommendations using rule-based logic only. Do not use external AI APIs or require API keys.
* Include an **"I'm Behind"** button that recalculates the schedule based on the remaining time.
* Dashboard should display today's tasks, upcoming deadlines, progress and insights.

## Code Quality

* Clean folder structure.
* Reusable components.
* Type-safe code.
* Proper loading, validation and error handling.
* Use environment variables where needed.
* Keep the code concise, readable and natural.
* Avoid unnecessary comments. and humanized code

## Behavior

* Make reasonable implementation decisions without asking unnecessary questions.
* Complete features end-to-end before moving on.
* Fix build and runtime errors encountered.
* Keep the UI modern, minimal and polished.
* Prefer the simplest, fastest and most reliable implementation.
* Avoid unnecessary complexity and over-engineering.
* Reuse existing libraries whenever appropriate.
* If multiple approaches exist, choose the one that minimizes development time while maintaining quality.
* Focus on delivering a working MVP before adding extra features.
* Ensure the application is runnable locally with clear setup instructions.
